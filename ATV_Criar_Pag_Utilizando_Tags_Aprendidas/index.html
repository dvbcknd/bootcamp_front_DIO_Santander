O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação usada para criar páginas da web. Ele estrutura o conteúdo, dizendo ao navegador como exibir textos, imagens, links, vídeos, formulários etc. É como se fosse o esqueleto do site.

Depois da atualização para o HTML 5 foi adicionado novas tags, que são tags semânticas. Elas ajudam na clareza e propósito do conteúdo, além de melhorar o SEO do site.
São elas:

<article> = Para escrever, por exemplo, um artigo

<nav> = É usado para a navegação do site, normalmente colocamos links

<header> = É usado para o cabeçalho do site, aonde normalmentecolocamos o logo e menus

<footer> = É usado para criar o rodapé do site

<section> = uma seção de conteúdo

<main> = conteúdo principal

O que é semântica em HTML?

A semântica em HTML se refere ao uso de tags que descrevem claramente o propósito do conteúdo. Essas tags ajudam navegadores, motores de busca e tecnologias assistivas (como leitores de tela) a entender melhor o conteúdo.

Por que a semântica importa?

Melhora a acessibilidade.
Melhora o SEO (otimização para motores de busca).
Facilita a manutenção e a colaboração no código.
Ajuda leitores de tela a interpretar a página corretamente.

Use tags semânticas sempre que possível. Só use <div> ou <span> quando não houver uma alternativa semântica apropriada. Use atributos como aria-label para melhorar ainda mais a acessibilidade.

O que são títulos em HTML?

Títulos são criados usando as tags h1 até h6 e são usados para organizar o conteúdo da página de forma hierárquica. Eles informam qual é o conteúdo mais importante e ajudam na navegação, tanto para os usuários quanto para ferramentas como o Google e leitores de tela.

Use apenas um h1 por página, mantenha a ordem hierárquica, use os títulos para dividir seções claramente e vite usar h1 - h6 apenas para mudar o tamanho do texto, use css para isso.

Tabelas em HTML

As tabelas são usadas para exibir dados organizados em linhas e colunas.
Tags importantes:

<table> = Inicia a tabela.

<tr> = Cria uma linha na tabela.

<th> = Cria o cabeçalho da coluna.

<td> = Cria as célula de dados da tabela.

<thead> = Aonde pode ser colocado o cabeçalho da coluna.

<tbody> = Aonde colocamos o corpo da tabela.

<tfoot> = Rodapé da tabela (opcional). Aonde colocamos, por exemplo, um campo de "Total", caso aja.

Formulários em HTML

Formulários são campos que criamos para que o usuário digite alguma informação solicitada, pois permitem que o usuário envie dados para o servidor (ou para um script).

Principais elementos de formulário:

<form> = Define o formulário.
action="" = É adicionado dentro da tag form e colocamos o caminho para onde os dados serão enviados.

method="get/post" = É colocado também dentro da tag form, é o método de envio (GET = URL, POST = corpo).

<input> = Por essa tag que inserimos os campos de entrada (texto, email, senha etc.).

<textarea> = Cria uma área de texto maior.

<select> / <option> = Usados para criar opções pré estabelecidas para o usuário.

<button> = Insere um botão (geralmente para enviar ou resetar).

<label> = Insere um rótulo do campo, vai antes do input.

required = É colocado dentro de um input para tonar aquele campo obrigatório.

placeholder = Gera um texto de exemplo dentro da guia do campo.
